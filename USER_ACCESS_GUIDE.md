## ğŸ› Troubleshooting

### Issue: Can't see the app

**Solution**:
1. Check URL is `http://localhost:5173`
2. Make sure backend is running
3. Wait a few seconds for page to load
4. Hard refresh: Ctrl+F5

### Issue: Login fails

**Solution**:
1. Check email is correct (case-sensitive)
2. Check password is correct (case-sensitive)
3. Verify account was created during registration
4. Try creating a new test account

### Issue: Pages look broken

**Solution**:
1. Hard refresh: Ctrl+F5
2. Clear browser cache
3. Open DevTools (F12) and check Console for errors
4. Try a different browser

### Issue: Buttons don't work

**Solution**:
1. Check internet connection
2. Verify backend is running (port 3000)
3. Check browser console for errors (F12)
4. Try refreshing the page

### Issue: Role-based menu not changing

**Solution**:
1. Logout and login again
2. Clear browser storage: DevTools â†’ Application â†’ Storage â†’ Clear All
